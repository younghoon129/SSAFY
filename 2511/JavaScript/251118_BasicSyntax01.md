# Basic Syntax01


## 데이터 타입
- 원시 자료형
    - 값 자체가 변수에 직접 저장
    - 불변, 변수 간 할당 시 값이 복사
    - 종류
        - Number
            - 문자열 + 숫자 시 숫자가 문자열 자동 형 변환
            - 정수, 실수 구분 x 
        - String
            - +로 문자끼리 결합
            - `${}` -> Fstring (ES6+ 부터 지원)
        - Boolean
            - Boolean 데이터 타입 아니어도 '자동 형변환' 으로 인해 true 또는 false로 변환됨
        - null
            - '의도적으로 값이 없음'을 나타냄
            - 타입: Object -> 0
        - undefined
            - '값이 할당되지 않음' 의도 x
            - 타입: undefined -> NaN
- 참조 자료형
    - 데이터가 저장된 메모리의 주소가 변수에 저장되는 자료형
    - 가변, 변수 간 할당 시 주소가 복사
    - Objects(Object, Array, Function)

## 연산자
- 종류
    - 할당 연산자
        - 단축 연산자 지원(-=, += 등)
    - 증가, 감소 연산자
        - 증가: ++
        - 감소: --
        - 가독성을 위해 단축 연산자 권장
    - 비교 연산자
        - 결과값 boolean으로 반환하는 연산자(>, <)
    - 동등 연산자(==)
        - boolean 값을 반환
        - 타입 상관 x (암묵적 타입 변환)
        - 같은 객체인지 확인
    - 일치 연산자(===)
        - 객체, 값, 타입 확인
        - 암묵적 타입 변환 X
    - 논리 연산자
        - and(&&)
        - or(||)
        - not(!)
        - 단축평가 지원
