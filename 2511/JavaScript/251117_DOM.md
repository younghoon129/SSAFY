# DOM
- 웹의 기술적 발전과 웹 표준의 중요성

## ECMAScript
- 표준화된 스크립트 프로그래밍 언어 명세
- javaScript는 ECMAsript 표준을 따르는 구체적인 프로그래밍 언어

## 변수 작성 규칙
- 식별자(변수명)
  - 카멜 케이스
    - 변수, 객체, 함수에 사용
  - 파스칼 케이스
    - 클래스, 생성자에 사용
  - 대문자 스네이크 케이스
    - 상수에 사용
- 변수 선언 키워드
  - let(블록 스코프)
    - 재할당 가능
    - 재선언 불가능
  - const(블록 스코프)
    - 재할당 불가능
    - 재선언 불가능
  - var
    - 호이스팅 문제로 사용 권장 안함
- const를 기본으로 사용할 것
  - 코드의 의도 명확화
  - 버그 예방
  - 필요한 경우에만 let으로 전환

## 문서 구조
- DOM
  - 웹 페이지를 구조화된 객체로 제공하여 프로그래밍 언어가 페이지 구조에 접근할 수 있는 방법을 제공
- DOM tree
  - 객체 간 상속 구조가 존재
- DOM 선택
  - 조작 순서
    1. 요소 선택(탐색)
    2. 선택된 요소의 콘텐츠 또는 속성을 조작

## 선택 메서드
- document.querySelector(selector)
  - 요소 한개 선택
  - 없다면 null 반환
- document.querySelectorAll(selector)
  - 여러개 선택

## DOM 조작
- 속성 조작
  - 클래스(classList)
    - HTML 요소의 클래스 목록을 쉽게 제어(추가/제거)하는 도구
    - element.classList.add()
    - element.classList.remove()
    - element.classList.toggle()
  - 일반 속성
    - Element.getAttribute()
      - 지정된 값 반환(조회)
    - Element.setAttribute()
      - 지정된 요소의 속성 값을 설정
      - 이미 있으면 기존 값을 갱신, 없으면 새로 추가
    - Element.removeAttribute()
- HTML 콘텐츠 조작
  - textContent
    - 순수한 텍스트 데이터만 얻고 싶을 때 유용
- DOM 요소 조작
  - document.createElement(tagName)
  - Node.appendChild()
    - 추가된 Node 반환
  - Node.removeChild()
    - 제거된 Node 반환

## 용어 정리
- 함수 스코프
  - 함수의 중괄호 내부를 가르킴
- 호이스팅
  - 변수 선언문이 코드의 최상단으로 끌어올려지는 듯한 현상
  - 할당 전까지는 undefined 값을 가짐
