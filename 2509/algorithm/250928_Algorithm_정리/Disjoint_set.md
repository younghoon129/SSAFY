# 서로소 집합
- 서로 중복 포함된 원소가 없는 집합들(교집합x)
- 두 집합 간에 공통 원소 없을 때 두 집합을 서로소 집합
- 대표자(집합에 속한 특정 노드로 구분)
- 상호 베타 집합 표현
    * 연결 리스트
      * 같은 집합의 원소들은 하나의 연결리스트로 관리
      * 맨 앞의 원소를 대표자
    * 트리
      * 자식 노드가 부모 노드 가르킴, 루트 노드가 대표자

- 서로소 집합 연산 효율 높이는 법
    * 서로소 집합 최적화
    * Path compression
      * find-set에서 포인터 바꾸기
    * Rank로 관리
