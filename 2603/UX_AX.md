# 네트워크 패킷 분석
## 와이어샤크
- 실시간으로 패킷을 캡쳐하고 분석하는 오픈소스 도구
- TCP/IP, HTTP, DNS, TLS 등 다양한 프로토콜 지원
- 웹 디버깅, API 트래픽 분석, 보안 점검, 성능 문제 해결 등에 활용
- 활용 방법
    - 프론트엔드: API 요청, WebSocket, 네트워크 성능 분석
    - 백엔드: 서버 응답 분석, DB 쿼리 모니터링, 보안 점검

## 패킷 보안
- 데이터 통신 패킷을 캡쳐하여 해당 패킷을 누르면 세부정보를 볼 수 있음
- http 통신으로 요청을 보낼 경우 로그인 정보같은 민감한 정보가 그대로 노출될 수 있음
- 반드시 https 통신을 이용해서 정보 토출을 방지할 것

## 기업환경에서 주용되는 주요 업무
- 네트워크 문제해결
- 보안 사고 대응
- 트래픽 모니터링및 성능 분석
- 침해 탐지 및 악성 트래픽 분석 애플리케이션 디버깅 및 API 분석

## tcpdump
- 네트워크를 통해 송/수신되는 패킷들을 확인할 수 있는 명령어
    - tcpdump [-i eth0] [src/dst] host <IP>
    - tcpdump [-i eth0] port <port>
    - tcpdump -i eth0 port 8080 -w <파일명>
