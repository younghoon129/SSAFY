## NPM
- Node Package Manager
- 프로젝트에 필요한 패키지(라이브러리)를 손쉽게 관리(설치, 업데이트, 삭제)할 수 있도록 도와주는 패키지 관리 도구
- yarn, pnpm

## Package Manager의 주요 기능
- Dependencies 관리
- 프로젝트 Metadata 관리
- 스크립트 실행
- 패키지 배포(NPM Registry)

## initializer 를 활용하여 프로젝트 생성
- npm create vue@latest
- npm create react-app@latest
- npm create vite@latest(가장 자주 쓰임, 번들러)
