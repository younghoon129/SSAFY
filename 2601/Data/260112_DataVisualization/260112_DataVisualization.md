- Plotly를 활용한 시각화

## 데이터 시각화
- 데이터 시각화의 필요성
    - 수치만으로 보기 어려운 걸 시각화하여 데이터 파악하는데 시간단축할 수 있음
- 알맞은 시각화 방식을 선택해야 하는 이유
    - 같은 데이터여도 시각화 방식에 따라 데이터를 파악하는 방법이 다름
    - 그래서 시각화 데이터를 볼 때 **의도가 담겨 있는 지** 확인해야 됨
- 종류
    - 히스토그램
        - 연속형 데이터의 도수 분포를 나타냄
        - X축은 특정 구간, Y축은 해당 구간에 속한 데이터 개수
        - 데이터의 분포 확인, 특정 값이 많이 나타나는지 분석
    - 바이올린 플롯
        - 박스 플롯과 KDE(커널 밀도 추정)을 결합한 형태로 데이터 분포를 더욱 직관적으로 표현
    - 박스 플롯
        - 데이터의 분포와 이상치를 시각화하는 그래프
        - 중앙값, 사분위수, 이상값을 표시
        - 데이터 분포 비교, 이상치 탐지
    - 산포도
    - 바 차트
        - 특정 범주별 평균 또는 합계를 비교
        - X축에 범주형 데이터, Y축에 연속형 데이터
        - 범주별 데이터 비교, 그룹별 평균값 또는 합계 비교
    - 라인 플롯
    - 파이 차트
    - 영역 차트
    - 산포도
        - 두 개의 연속형 변수 간 관계를 나타내는 산점도, 점들이 특정 패턴을 따르는지 확인 가능
        - 변수 간 관계 분석, 상관관계 확인
        - 데이터 분포 패턴 분석
    - 선 그래프
        - 시계열 데이터를 시각화하는 그래프
        - X축에 시간요소, Y축에 연속형 데이터를 배치
        - 시간에 따른 데이터 변화 분석, 추세 파악
    - 방사형 차트
    - 버블 차트
        - 산포도와 유사
        - 점의 크기를 추가적인 변수를 통해 표현
        - 다차원 데이터 시각화, 변수 간 관계 분석
    - 깔때기 차트
    - 히트맵
        - 행렬 형태의 데이터를 색상으로 표현하여 관계를 시각화
        - 상관관계 분석, 행렬 데이터 시각화
    

## Matplotlib
- 교재 참고
- Matplotlib
    - 장점
        - 유연성: 모든 종류의 플롯 가능, 세밀한 커스터마이징
        - 넓은 사용자층과 문서: 풍부한 자료와 커뮤니티 지원
        - 정교한 제어: 그래프의 모든 요소 조정 가능
    - 단점
        - 복잡성: 간단한 작업도 많은 코드 필요
        - 기본 스타일: 추가 스타일링이 필요할 수 있음
- pyplot의 중요 요소
    - Figure
        - 그림을 그리기 위한 Canvas 역할
        - 그림판의 크기 등을 조절
    - Axes
        - 실제 그림을 그리는 메소드들을 가짐
        - x축, y축, Title 등의 속성 설정
- 여러 구성 요소 활용
    - color
    - marker
    - linestyle
    - linewidth
    - marksize

## Seaborn
- 장점
    - 간결함: 적은 코드로 세련된 시각화 가능
    - 기본 스타일: 매력적인 기본 스타일 제공
    - 데이터프레임 친화적: Pandas와 잘 통합됨
    - 고급 통계 기능: 통계적 시각화 쉽게 구현
- 단점
    - 제한된 커스터마이징: 세부적인 조정 어려움
    - 유연성 부족: 특정 그래프나 기능에 한계
- Figure level 함수, Axes level 함수
    - Figure-level 함수는 Figure 전체를 관리
    - Axes-level 함수는 개별 Axes에서 하나의 플롯을 그리는 데 사용
    - 판단기준: 비교용 그래프가 여러 개인가?

## Plotly
- 장점
    - 상호작용성: 상호작용 가능한 그래프 기본 제공
    - 웹 통합: 웹 기반 시각화에 최적화
    - 3D 및 지리적 시각화: 고급 시각화를 간단히 구현
    - 시각적 미학: 세련된 기본 그래프 제공
- 단점
    - 학습 곡선: 고급 기능 활용을 위해 학습 필요
    - 대규모 데이터 처리 한계: 성능 문제 발생 가능
    - 복잡한 커스터마이징: 세밍한 조정이 복잡할 수 있음
