## 📜 서술형 문제 (5문제)

**문제 8.** '정확도(Accuracy)'가 '불균형 데이터'(예: 99% 정상, 1% 불량)에서 평가지표로 부적절한 이유를, '모든 것을 정상으로 예측하는 모델'의 예시를 들어 설명하시오.

**문제 9.** '정밀도(Precision)'와 '재현율(Recall)'의 정의(공식)를 각각 TP, FP, FN, TN을 사용하여 서술하시오.

**문제 10.** **(1) 스팸 메일 필터**와 **(2) 암 진단 모델**의 두 가지 시나리오가 있습니다. 각각의 경우 '정밀도'와 '재현율' 중 **어떤 지표가 더 중요하게 고려되어야 하는지**, 그리고 **그 이유를 FP와 FN의 치명도와 연관 지어** 설명하시오.

**문제 11.** '조화 평균(F1 Score)'이 '산술 평균' 대신 사용되는 이유를, **(1) 정밀도=0.9, 재현율=0.1**인 모델 A와 **(2) 정밀도=0.5, 재현율=0.5**인 모델 B의 '산술 평균'과 '조화 평균'의 차이를 예시로 들어 설명하시오. (계산 포함)

**문제 12.** 다음 혼동 행렬을 보고 **(1) 정확도, (2) 정밀도, (3) 재현율**을 각각 계산하시오.
* **TP = 50**
* **FN = 50**
* **FP = 10**
* **TN = 90**

### 📝 단답형 / 서술형 예시 답안

> **단답형 정답**
> 1.  **MSE (평균 제곱 오차)**
> 2.  **정확도 (Accuracy)**
> 3.  **혼동 행렬 (Confusion Matrix)**
> 4.  **정밀도 (Precision)**
> 5.  **재현율 (Recall)**
> 6.  **정밀도 (Precision)** (FP가 분모에 있으므로)
> 7.  **재현율 (Recall)** (FN이 분모에 있으므로)
>
> ---
>
> **서술형 정답 (예시)**
> 8.  99%가 정상, 1%가 불량인 데이터에서, 모델이 **'모든 것을 정상'**이라고만 예측해도 100개 중 99개를 맞추므로 **정확도는 99%**가 나옵니다. 하지만 이 모델은 정작 중요한 '불량'을 단 하나도 찾아내지 못하는 '쓸모없는 모델'입니다. 이처럼 정확도는 불균형 데이터에서 다수 클래스의 성능만 반영하고 소수 클래스의 성능을 왜곡합니다.
>
> 9.
> * **정밀도(Precision)** = $TP / (TP + FP)$
> * **재현율(Recall)** = $TP / (TP + FN)$
>
> 10.
> * **(1) 스팸 메일 필터**: **정밀도**가 더 중요합니다. 실제 '정상' 메일(N)을 '스팸'(P)으로 잘못 예측하면(**FP**), 중요한 메일을 놓치게 되어 치명적입니다. 정밀도는 이 FP를 낮추는 것이 목표입니다.
> * **(2) 암 진단 모델**: **재현율**이 더 중요합니다. 실제 '암환자'(P)를 '정상'(N)으로 잘못 예측하면(**FN**), 치료 시기를 놓쳐 생명이 위험해지므로 치명적입니다. 재현율은 이 FN을 낮추는(하나도 놓치지 않는) 것이 목표입니다.
>
> 11.
> * **산술 평균**: 모델 A = $(0.9+0.1)/2 = 0.5$, 모델 B = $(0.5+0.5)/2 = 0.5$. (두 모델이 동일해 보임)
> * **조화 평균(F1)**: 모델 A = $2*(0.9*0.1)/(0.9+0.1) = 0.18$, 모델 B = $2*(0.5*0.5)/(0.5+0.5) = 0.5$.
> * **이유**: 산술 평균은 두 값의 '균형'을 보지 못하지만, **조화 평균은 두 값 중 '낮은 값'에 더 큰 페널티를 부여**합니다. 따라서 정밀도와 재현율이 '불균형'(모델 A)하면 점수가 매우 낮아지고, '균형' 잡힌(모델 B) 모델을 더 높게 평가하므로 F1 점수를 사용합니다.
>
> 12.
> * **(1) 정확도 (Accuracy)** = $(TP+TN) / (전체)$ = $(50+90) / (50+50+10+90)$ = $140 / 200$ = **0.7 (70%)**
> * **(2) 정밀도 (Precision)** = $TP / (TP+FP)$ = $50 / (50+10)$ = $50 / 60$ = **약 0.833 (83.3%)**
> * **(3) 재현율 (Recall)** = $TP / (TP+FN)$ = $50 / (50+50)$ = $50 / 100$ = **0.5 (50%)**
