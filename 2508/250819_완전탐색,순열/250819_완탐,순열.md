## 순열
- 개요(Permutation)
    * N!/(N-r)! -> r개를 뽑는다 했을 때 경우의 수
    * 서로 다른 것 중 몇 개를 뽑아서 순서대로 나열하는 것
        * 줄 세우기, 대회에서 순위
    * 서로 다른 n개 중 r 개를 택하는 순열은
        * nPr 로 표현
    * nPr은 다음과 같은 식이 성립
        * nPr = n*(n-1)*(n-2)...*(n-r+1)
    * nPn = n! = n*(N-1)*(n-2)
 
    * 순서화된 요소들의 집합에서 최선의 방법을 찾는 것과 관련 있음
        * N개가 12개 이상이다? 그럼 순열 쓰면 안됨 (시간복잡도 12!)

- 구현 : 반복문
    * for i (1,4):
        * for j (1, 4):
            if i != j
                for k (1, 4)
                    if i != k
                        print(i,j,k)
